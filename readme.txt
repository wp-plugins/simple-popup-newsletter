== Description ==
The program manages the newsletter sign-ups and unsubscibtions with the help of a pop-up window. The pop-up window will appear at the top of the web page with an animation and it will be waiting for the visitor interaction.

= Shortcode =

If you want to place the buttons, use these shortcodes.

* [subscribtion caption="newsletter subscribtion"]
* [subscribtion caption="newsletter unsubscribtion"]


== Installation ==

1. install and activate the plugin on the Plugins page
2. set the language and other options
3. add shortcodes to pages: [subscribtion], [unsubscribtion]
4. use popups windows of the program to collect email addresses
